### Automatically sends emails from .CSV files exported from CrunchBase

Instructions:
1. Install Visual Studio Code for your operating system: https://code.visualstudio.com/download
2. Open Visual Studio Code.
3. Click the **_Extensions_** button on the left sidebar and look up "Python". Install and enable the Python extension.

4. Navigate to **_Explorer_** on the left sidebar and select **_Clone Repository_**.
<img width="132" alt="image" src="https://github.com/Aanika-Sadana/EmailSender/assets/70586980/09e1b68f-95b7-471d-a94a-58b4d21600d5">

5. Enter the following repository URL in the above text box: https://github.com/Aanika-Sadana/EmailSender.git
